Parameters (1/2)

Mandatory parameters

First, you may calculate a daily infection rate among your staff by using our Infection Rate Estimation Tool.

Shift length relates to the shift pattern. If your shifts are based around the 7-day working week then enter shift length as 7, even if you are not working for 7 days straight. You can then enter the number of days off per shift later.

The number of areas can relate to physical areas e.g. covering two wards would be equal to 2. Or it might relate to different shifts e.g. day, evening and night equal to 3.

If you have not calculated a daily infection rate, then you can enter value of your choosing for the off-work period. See the box below for what happens with different daily rates over time. Optional parameters

If you have a critical area you can specify which area that is here. You can only have one critical area. The number that you enter tells the model which area is critical e.g. if the first area, enter 1, if its the second area, enter 2. Also enter the critical number of staff (the minimum number of staff required) for that critical area. Both parameters must be provided at the same time. If you do not have them leave None.
